# Nicky Spa Therapy

A luxury spa website built with React and Vite for Nicky Spa Therapy in Kigali.

## Folder Structure

- public/ for static assets such as favicon, logo, and images
- src/assets/ for hero, gallery, services, testimonials, and about images
- src/components/ for shared UI pieces such as navbar, footer, buttons, cards, and social links
- src/pages/ for home, about, services, packages, gallery, booking, contact, and not-found views
- src/data/ for page content and reusable data
- src/styles/ for global styling and theme variables

## Development

Run the app locally:

```bash
npm install
npm run dev
```
